# AndroidBandBridge

Aplikacja BandBridge for Android pozwala na gromadzenie danych HR i GSR z opasek Microsoft Band 2 podłączonych do urządzenia z systemem Android.

Jest częścią projektu realizowanego w ramach pracy magisterskiej na wydziale Fizyki, Astronomii i Informatyki Stosowanej Uniwersytetu Jagiellońskiego w Krakowie.
